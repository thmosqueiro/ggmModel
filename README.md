# ggmModel


# Usage

You can simply run the following.

```
make
time ./ggmModel $RANDOM 10000 10. output.dat
```

This will construct a network and output all degrees in output.dat.

Sample output:
```
[user@PC C++]$ time ./ggmModel $RANDOM 10000 10. output.dat

real	0m0.418s
user	0m0.102s
sys	0m0.316s
```
